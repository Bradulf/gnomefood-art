# Generated Prompts

Final prompts go here after `story.md`, `traversal.md`, and `layers/*.md` are written.

All generated strip prompts must preserve the shared camera/canvas alignment: same full-width canvas size, same side-view camera height and angle, same camera framing, same horizon placement, same bottom alignment, same camera distance, and same world scale.

All strip assets must be transparent PNG compositing strips with clean alpha edges, no rectangular backdrop, no sky plate, no cloud plate, no horizon background fill, no white background, no black background, and no full scene backdrop.
