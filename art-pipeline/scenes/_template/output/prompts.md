# Generated Prompts

Final visual chronicle prompts go here after `story.md`, `traversal.md`, `layers/*.md`, and `output/shared-context.md` are written.

Each strip prompt must be fully self-contained. Do not rely on scene-wide context above the prompts; copy the compact shared context into every prompt block.

```text
Generation command:
Create one transparent PNG image for this single compositing strip.

Strip: 01-front-strip - front-strip
```
